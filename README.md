## Мобільний додаток "Кривий Ріг - Туристичний"

Мета проекту: створити реальну і вкрай необхідну практичну платформу для поширення інформації про туристичні об’єкти і маршрути Кривого Рогу, збільшить кількість туристів і зробить наш регіон більш доступним для туристів, а також створить умови для освіти і виховання, зростання інтелектуального потенціалу молоді та усього населення міста.

_Кривий Ріг, 2020_

## Install and run:

1. `cp .env.sample .env` and fill it in with the required credentials
2. `yarn install`
3. `yarn start`
4. Navigate to [http://localhost:3000/](http://localhost:3000/)
